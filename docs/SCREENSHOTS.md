# Screenshots

## ARIA World Dashboard

![ARIA World dashboard preview](screenshots/world_dashboard_preview.svg)

Generate the live dashboard:

```powershell
python run_dashboard.py --days 60 --agents 12 --seed 42
```

Output:

```text
docs/screenshots/world_dashboard.html
```

The generated HTML includes animated agents, trust and knowledge graphs, resource levels, culture, innovations, and benchmark scores.
