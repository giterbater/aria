# ARIA Wiki Overview

## What Is ARIA?

ARIA is a modular AI research system. It separates language, cognition, memory, execution, learning, benchmarking, and simulation into inspectable subsystems.

## What Is ARIA World?

ARIA World is a civilization-style simulation where agents live, work, trade, learn, reproduce, form culture, and discover innovations.

The world is useful because it gives ARIA a measurable environment. Instead of only reading logs, contributors can watch population, trust, knowledge, resources, and benchmarks evolve.

## Important Concepts

| Concept | Meaning |
|---|---|
| Agent | A simulated villager powered by ARIA subsystems |
| Trust network | Social cohesion between agents |
| Knowledge graph | Village knowledge and expertise distribution |
| Culture | Customs and proven strategies |
| Innovation | Recipes and discoveries |
| Benchmark score | Quantified simulation health |

## Where To Start

1. Generate the dashboard with `python run_dashboard.py`.
2. Read `docs/architecture/world.md`.
3. Run `tests/test_aria_world.py`.
4. Compare benchmark scores after changes.
